## Instalação

- Clone o repositório do projeto para uma pasta local

- Dentro da pasta **arquivos**, ou uma outra com o nome que preferir, insira os arquivos a serem utilizados. Essa pasta funcionará como o diretório de arquivos

- No arquivo **config.py**:
    - Insira os arquivos default na lista de arquivos declarada

    - Configure a porta do diretório raiz (preferencialmente 80 ou 8080)

    - Atribua o nome do arquivo da página de erro para a variável **pagErro**

    - Atribua o nome do diretório de arquivos para a variável **dir**

- Para inicializar o servidor, basta digitar no terminal o comando **“python3 server.py”**
